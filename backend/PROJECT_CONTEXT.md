# SlashDR

## Project Overview

SlashDR is an internal clinic management system.

The project is NOT a marketing website.

Its goal is to help clinic staff manage:

- Consent Templates
- Consent Records
- Clinic Licenses
- Uploaded Documents
- Audit Logs

Backend is approximately 75% complete.

Frontend has not yet been developed.

---

## Technology Stack

Backend
- Spring Boot
- Java
- Spring Security
- Spring Data JPA
- Thymeleaf
- MySQL

Frontend
- HTML
- CSS
- JavaScript

IMPORTANT

Do NOT migrate frameworks.

Do NOT use React.

Do NOT use Angular.

Do NOT use Vue.

Do NOT rewrite the backend.

Do NOT change API contracts.
